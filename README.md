## 前端与算法部分

数据结构三分学,七分练,重在自己实现做了,但是我个人更同意一个观点,如果是有目的的刷,第一遍尽量先看别人的题解,了解了别人的方法,你才能更加快速的进步,很多时候知道有这个方法比自己摸索出这个办法更有价值,就算是去探索知识的边界,也要了解了很多事情才能去探索,做算法也是这个道理,站在前人的肩膀上走更远,才是我们的目标
